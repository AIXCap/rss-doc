# 管理聚合源（Poly Feed）

## 管理聚合源

1. 在 `RssTabs` 侧边栏右键您的 `Poly Feed` 即可从下拉菜单中选择管理您的源
2. 点击 `管理` 按钮即可进入管理或编辑界面
3. 同 [创建 Poly Feed](./create-poly-feed.md) 中的步骤类似，您可以在这里编辑并提交您的 `Poly Feed`
4. 请注意，您订阅的 `Poly Feed` 不可编辑
5. 同样的，您创建的公开 `Poly Feed` 的所有订阅者将会同步您对此源的更新
