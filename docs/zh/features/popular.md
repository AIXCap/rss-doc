# 源广场

## 简介

`RssTabs` 提供了一个全面的订阅源管理系统，即源广场，让你可以发现、订阅和组织来自网络各处的 `Poly Feed`。

源广场是你发现新内容的门户。在这里你可以：

- 浏览热门 `Poly Feed`
- 获取个性化 `Poly Feed` 推荐
- 订阅公开、热门的 `Poly Feed`
- 复制公开、热门的 `Poly Feed`
